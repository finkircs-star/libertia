Privacy Policy for Libertia

Last Updated: January 22, 2025

Overview

Libertia is a browser extension designed to help users block gambling websites and track their recovery progress. We are committed to protecting your privacy and being transparent about our data practices.

Data Collection

Libertia does NOT collect, transmit, store, or share ANY user data with external servers or third parties.

All data generated by the extension is stored locally on your device using Chrome's built-in storage API.

What Data Is Stored Locally

The following data is stored locally on your device and never leaves your browser:

Protection Settings

  • Extension enabled/disabled state
  • Optional Gemini API key (if you choose to use AI detection)

Statistics

  • Number of gambling sites blocked
  • Time saved (calculated locally)
  • Money saved and lost (user-entered amounts)
  • Event history (save/loss transactions)

Progress Tracking

  • Clean streak (days without gambling)
  • Best streak record
  • Achievement badges earned
  • Last gambling event timestamp

No Personal Information

We do NOT collect, store, or process:

  • Your name, email, or contact information
  • Browser history (beyond checking URLs against our blocklist)
  • Personal identification data
  • Location data
  • Payment information
  • Cookies or tracking pixels

How We Use Local Data

Data stored locally is used solely to:

  1. Block gambling websites based on our domain and keyword lists
  2. Track your recovery progress (streaks, money saved, achievements)
  3. Provide visual charts and statistics
  4. Enable data export for your personal records

Third-Party Services

Optional: Gemini AI Detection

If you choose to enable AI-powered detection:

  • You must provide your own free Gemini API key from Google
  • The API key is stored locally in your browser only
  • We never see, store, or have access to your API key
  • When AI detection is active, page content may be sent to Google's Gemini API for classification
  • This is a direct connection from your browser to Google - no data passes through our servers
  • You can disable AI detection and remove your API key at any time

We do not operate any servers or databases. All functionality runs locally in your browser.

Data Sharing

Libertia does NOT:

  • Share your data with third parties
  • Sell your data to advertisers
  • Track your behavior across websites
  • Use analytics or telemetry
  • Display advertisements

The only external connection is the optional Gemini API call (if you enable it and provide your own key).

Data Export

The extension includes an "EXPORT DATA" feature that:

  • Creates a JSON file of your local statistics
  • Downloads directly to your device
  • Never uploads to any server
  • Is entirely under your control

Data Deletion

You can delete all your data at any time:

  • Click "RESET DATA" in the extension popup to delete all statistics
  • Uninstall the extension to remove all local data
  • Remove your Gemini API key from settings if you provided one

Permissions Explained

storage

Why: Save your settings and statistics locally on your device
Data Access: Local browser storage only

tabs

Why: Detect when you visit gambling sites to block them
Data Access: URL of current tab only, no browsing history

webNavigation

Why: Monitor page navigation to check URLs against our blocklist
Data Access: Navigation events for blocking purposes only

scripting

Why: Extract page content for optional AI analysis (only if you enable it)
Data Access: Page title and text content, only when AI detection is active

alarms

Why: Update time saved counter and streak calculations
Data Access: None - just a timer

host_permissions (<all_urls>)

Why: Check any visited URL against our gambling site blocklist
Data Access: URL comparison only, no data collection

Children's Privacy

Libertia is intended for users age 18 and over. We do not knowingly collect data from children under 18.

Security

  • All data is stored locally using Chrome's secure storage API
  • No server-side storage or databases
  • No network connections except optional Gemini API (if enabled)
  • Open source code available for review

Changes to This Policy

We may update this privacy policy from time to time. Changes will be reflected in the "Last Updated" date at the top of this document.

International Users

Libertia works the same way for all users regardless of location:

  • All data is stored locally on your device
  • No data is transmitted to any country or jurisdiction
  • GDPR, CCPA, and other privacy laws are respected by design (no data collection = no compliance issues)

Your Rights

Since Libertia does not collect or store your data on any servers, you have complete control:

  • Access: Your data is in your browser's local storage
  • Export: Use the "EXPORT DATA" button to download your statistics
  • Delete: Use the "RESET DATA" button or uninstall the extension
  • Portability: Export function provides your data in standard JSON format
